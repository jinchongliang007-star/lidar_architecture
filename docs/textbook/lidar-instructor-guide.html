LiDAR 教学教师手册

课程概述

课程定位

本课程为汽车专业和新能源汽车技术专业学生设计的 LiDAR 技术入门课程，通过理论和实践相结合的方式，帮助学生理解机械旋转式激光雷达的工作原理。

课程目标

完成本课程后，学生应能够：

  1. 知识层面
  2. 理解 LiDAR 的基本工作原理
  3. 掌握点云数据的生成过程

  4. 了解射线碰撞检测算法

  5. 技能层面

  6. 能够运行和操作 LiDAR 演示程序
  7. 能够调整参数并分析对点云质量的影响

  8. 能够阅读和修改基础 Python 代码

  9. 应用层面

  10. 理解 LiDAR 在自动驾驶中的应用
  11. 了解不同场景下的参数选择策略

第一章：教学概述

教学目标：让学生明确学习目标，了解先修知识要求

重点内容： 1. LiDAR 在自动驾驶中的重要性 2. 课程的学习路径 3. 先修知识自测

教学建议： - 播放自动驾驶汽车感知系统视频 - 展示 LiDAR 点云可视化效果 - 进行简单的数学基础测试（向量、三角函数）

常见问题： - Q: 这门课需要很强的编程基础吗？ - A: 只需要能阅读 Python 代码即可，不要求独立编写复杂程序

第二章：LiDAR 技术背景

教学目标：建立 LiDAR 技术的基础认知

重点内容： 1. 激光测距基本原理（飞行时间法） 2. 机械旋转式 LiDAR 的结构组成 3. 点云数据格式和应用场景

教学建议： - 使用实物或 3D 模型展示 LiDAR 结构 - 演示不同厂商的 LiDAR 产品（Velodyne、禾赛等） - 展示真实的点云数据示例

关键公式：

距离 = (光速 × 飞行时间) / 2

常见问题： - Q: 为什么除以 2？ - A: 因为激光是往返的，飞行时间是来回的总时间

  • Q: 固态 LiDAR 和机械式有什么区别？
  • A: 固态 LiDAR 没有旋转部件，通过电子扫描改变方向

补充资料： - Velodyne VLP-16 数据手册 - 禾赛 Pandar40P 技术规格

第三章：系统架构与设计

教学目标：理解演示系统的设计思路

重点内容： 1. 技术选型理由（PyQt5 + PyOpenGL） 2. 模块划分和职责 3. 数据流设计

教学建议： - 画出系统架构图，讲解各模块关系 - 使用类比帮助学生理解： - Scene = 舞台 - Camera = 摄像机 - Environment = 道具 - 强调模块化设计的重要性

常见学生问题解答

原理类问题

Q1: LiDAR 为什么在夜间也能工作？

A: LiDAR 使用的是主动激光光源，不依赖环境光，所以夜间也能正常工作。这是 LiDAR 相比摄像头的优势之一。

Q2: 为什么 LiDAR 点云有些地方是空的？

A: 可能的原因有： 1. 该区域没有物体反射（如天空） 2. 物体表面吸收激光（黑色吸光材料） 3. 物体距离超过探测范围 4. 物体在盲区内（垂直 FOV 之外）

Q3: RPM 越高越好吗？

A: 不是。RPM 越高： - 优点：更新频率高，对动态物体响应快 - 缺点：点云密度降低，分辨率下降

需要根据应用场景平衡选择。

Q4: 为什么垂直方向的激光线分布可以是不均匀的？

A: 有些 LiDAR 设计会在线束靠近地面的区域分布更密集，因为： - 道路上的障碍物主要在地面附近 - 近距离检测需要更高精度 - 可以优化点云分布效率

编程类问题

Q5: Python 代码中的 self 是什么意思？

A: self 代表类的实例对象本身。在类的方法中，用 self.xxx 访问实例的属性和方法。

class Scene:
    def __init__(self):
        self.rotation_angle = 0  # 实例属性

    def update(self, dt):
        self.rotation_angle += 1  # 访问实例属性

Q6: glBegin 和 glEnd 之间是什么？

A: 这是 OpenGL 的立即模式，glBegin 和 glEnd 之间定义要绘制的图元（点、线、三角形等）：

glBegin(GL_POINTS)  # 开始绘制点
glVertex3f(x, y, z)  # 定义一个点
glVertex3f(x, y, z)  # 定义另一个点
glEnd()  # 结束绘制

Q7: 什么是归一化向量？

A: 归一化是将向量的长度变为 1，但保持方向不变：

import numpy as np

def normalize(v):
    length = np.linalg.norm(v)
    if length > 0:
        return v / length
    return v

在射线追踪中，方向向量通常需要归一化。

实操类问题

Q8: 程序运行报错 "OpenGL not available" 怎么办？

A: 1. 检查显卡驱动是否最新 2. 确认显卡支持 OpenGL 3.0+ 3. 尝试更新 PyOpenGL: pip install --upgrade PyOpenGL

Q9: 程序很卡怎么办？

A: 可能的优化方法： 1. 减少点云数量（增大水平分辨率） 2. 减少激光线数 3. 检查是否有其他程序占用 GPU

Q10: 如何在代码中添加新的物体？

A: 参考 environment.py 中 Vehicle 和 Wall 类的实现：

  1. 定义新类，包含 __init__、render、ray_intersect 方法
  2. 在 Environment.__init__ 中创建实例
  3. 在 Environment.ray_cast 中添加碰撞检测
  4. 在 Environment.render 中添加渲染
