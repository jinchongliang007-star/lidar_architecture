实验二：参数调节实验

实验目的

  1. 理解转速（RPM）对点云密度的影响
  2. 掌握激光线数与垂直覆盖范围的关系
  3. 观察垂直视场角（FOV）对扫描效果的影响

实验器材

  • 已配置完成的 LiDAR 演示程序

实验内容

2.1 转速实验

实验原理：转速（RPM）决定 LiDAR 每秒旋转的圈数，影响水平方向的采样密度。

实验步骤：

  1. 启动程序，将激光线数设为 16
  2. 将垂直 FOV 设为 30°
  3. 分别设置转速为 5、10、15、20 RPM
  4. 观察并记录点云密度变化

数据记录表：

转速 (RPM) 每圈水平点数（估算） 点云密度观察 旋转一周时间
5 密集 / 适中 / 稀疏 秒
10 密集 / 适中 / 稀疏 秒
15 密集 / 适中 / 稀疏 秒
20 密集 / 适中 / 稀疏 秒

思考题： - 为什么转速越高，点云越稀疏？ - 在实际自动驾驶中，如何平衡转速和点云密度？

2.2 激光线数实验

实验原理：激光线数决定垂直方向的采样密度，影响对高度变化的检测能力。

实验步骤：

  1. 将转速固定为 10 RPM
  2. 将垂直 FOV 固定为 30°
  3. 分别设置激光线数为 4、8、16、32
  4. 观察点云在垂直方向的分布变化

数据记录表：

激光线数 垂直角度间隔 地面检测效果 车辆轮廓清晰度
4 ° 差 / 中 / 好 差 / 中 / 好
8 ° 差 / 中 / 好 差 / 中 / 好
16 ° 差 / 中 / 好 差 / 中 / 好
32 ° 差 / 中 / 好 差 / 中 / 好

计算公式：

垂直角度间隔 = 垂直FOV / (激光线数 - 1)

2.3 垂直视场角实验

实验原理：垂直 FOV 决定 LiDAR 能够扫描的垂直范围。

实验步骤：

  1. 将转速固定为 10 RPM
  2. 将激光线数固定为 16
  3. 分别设置垂直 FOV 为 15°、30°、45°、60°
  4. 观察地面和高处物体的检测情况

数据记录表：

垂直 FOV 地面点云覆盖 车辆顶部检测 适用场景
15° 差 / 中 / 好 差 / 中 / 好
30° 差 / 中 / 好 差 / 中 / 好
45° 差 / 中 / 好 差 / 中 / 好
60° 差 / 中 / 好 差 / 中 / 好

4.2 统计点云分布

任务：统计不同类型物体的点云数量

# 添加统计代码
def analyze_point_cloud(self):
    stats = {'vehicle': 0, 'wall': 0, 'ground': 0, 'unknown': 0}
    for point in self.point_cloud_data:
        hit_type = point[3] if len(point) > 3 else 'unknown'
        stats[hit_type] = stats.get(hit_type, 0) + 1
    return stats

4.3 导出点云数据

任务：将点云数据导出为 CSV 文件

import csv

def export_point_cloud(self, filename='point_cloud.csv'):
    with open(filename, 'w', newline='') as f:
        writer = csv.writer(f)
        writer.writerow(['x', 'y', 'z', 'type'])
        for point in self.point_cloud_data:
            writer.writerow(point)
    print(f"导出 {len(self.point_cloud_data)} 个点到 {filename}")
