# LiDAR 教学教师手册

## 课程概述

### 课程定位

本课程为汽车专业和新能源汽车技术专业学生设计的 LiDAR 技术入门课程，通过理论和实践相结合的方式，帮助学生理解机械旋转式激光雷达的工作原理。

### 课程目标

完成本课程后，学生应能够：

1. **知识层面**
   - 理解 LiDAR 的基本工作原理
   - 掌握点云数据的生成过程
   - 了解射线碰撞检测算法

2. **技能层面**
   - 能够运行和操作 LiDAR 演示程序
   - 能够调整参数并分析对点云质量的影响
   - 能够阅读和修改基础 Python 代码

3. **应用层面**
   - 理解 LiDAR 在自动驾驶中的应用
   - 了解不同场景下的参数选择策略

### 第二章：LiDAR 技术背景

**教学目标**：建立 LiDAR 技术的基础认知

1. 激光测距基本原理（飞行时间法）
2. 机械旋转式 LiDAR 的结构组成

**教学建议**：
- 使用实物或 3D 模型展示 LiDAR 结构
- 演示不同厂商的 LiDAR 产品（Velodyne、禾赛等）

**关键公式**：
```
距离 = (光速 × 飞行时间) / 2
```

**常见问题**：
- Q: 为什么除以 2？
- A: 因为激光是往返的，飞行时间是来回的总时间

- Q: 固态 LiDAR 和机械式有什么区别？
- A: 固态 LiDAR 没有旋转部件，通过电子扫描改变方向

**补充资料**：
- Velodyne VLP-16 数据手册
- 禾赛 Pandar40P 技术规格

## 常见学生问题解答

### 原理类问题

**Q1: LiDAR 为什么在夜间也能工作？**

A: LiDAR 使用的是主动激光光源，不依赖环境光，所以夜间也能正常工作。这是 LiDAR 相比摄像头的优势之一。

**Q2: 为什么 LiDAR 点云有些地方是空的？**

A: 可能的原因有：
1. 该区域没有物体反射（如天空）
2. 物体表面吸收激光（黑色吸光材料）
3. 物体距离超过探测范围
4. 物体在盲区内（垂直 FOV 之外）

**Q3: RPM 越高越好吗？**

A: 不是。RPM 越高：
- 优点：更新频率高，对动态物体响应快
- 缺点：点云密度降低，分辨率下降

需要根据应用场景平衡选择。

**Q4: 为什么垂直方向的激光线分布可以是不均匀的？**

A: 有些 LiDAR 设计会在线束靠近地面的区域分布更密集，因为：
- 道路上的障碍物主要在地面附近
- 近距离检测需要更高精度
- 可以优化点云分布效率

### 编程类问题

**Q5: Python 代码中的 `self` 是什么意思？**

A: `self` 代表类的实例对象本身。在类的方法中，用 `self.xxx` 访问实例的属性和方法。

```python
class Scene:
    def __init__(self):
        self.rotation_angle = 0  # 实例属性

    def update(self, dt):
        self.rotation_angle += 1  # 访问实例属性
```

**Q6: `glBegin` 和 `glEnd` 之间是什么？**

A: 这是 OpenGL 的立即模式，`glBegin` 和 `glEnd` 之间定义要绘制的图元（点、线、三角形等）：

```python
glBegin(GL_POINTS)  # 开始绘制点
glVertex3f(x, y, z)  # 定义一个点
glVertex3f(x, y, z)  # 定义另一个点
glEnd()  # 结束绘制
```

**Q7: 什么是归一化向量？**

A: 归一化是将向量的长度变为 1，但保持方向不变：

```python
import numpy as np

def normalize(v):
    length = np.linalg.norm(v)
    if length > 0:
        return v / length
    return v
```

在射线追踪中，方向向量通常需要归一化。

### 实操类问题

**Q8: 程序运行报错 "OpenGL not available" 怎么办？**

A:
1. 检查显卡驱动是否最新
2. 确认显卡支持 OpenGL 3.0+
3. 尝试更新 PyOpenGL: `pip install --upgrade PyOpenGL`

**Q9: 程序很卡怎么办？**

A: 可能的优化方法：
1. 减少点云数量（增大水平分辨率）
2. 减少激光线数
3. 检查是否有其他程序占用 GPU

**Q10: 如何在代码中添加新的物体？**

A: 参考 `environment.py` 中 `Vehicle` 和 `Wall` 类的实现：

1. 定义新类，包含 `__init__`、`render`、`ray_intersect` 方法
2. 在 `Environment.__init__` 中创建实例
3. 在 `Environment.ray_cast` 中添加碰撞检测
4. 在 `Environment.render` 中添加渲染
